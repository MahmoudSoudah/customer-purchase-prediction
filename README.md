# Customer Purchase Prediction

## Project Description
This project predicts whether a customer will purchase a product based on Age and Estimated Salary using Logistic Regression.

## Model
- Logistic Regression
- Binary Classification (0 = No, 1 = Yes)

## Dataset
- Age
- Estimated Salary
- Purchased

## Features
- Data cleaning
- Model training
- Accuracy evaluation
- Data visualization
- Streamlit web application

## How to Run
1. Install libraries:
   pip install pandas matplotlib scikit-learn streamlit

2. Run the app:
   streamlit run app.py

## Result
The model predicts customer purchase behavior with good accuracy.
